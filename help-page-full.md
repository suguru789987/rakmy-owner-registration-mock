# 従業員アプリ（打刻者向け）

## パスワードを忘れた

ログイン画面の「パスワードを忘れた」より再設定が可能です。

> **📋 スクリーンショット:** 従業員ログイン画面 + パスワードを忘れた場合画面（既存スクショ流用）

1. ログイン画面右下の「パスワードを忘れた」を選択
2. 登録されているメールアドレスを入力
3. メールアドレス宛にリセット案内が届く

---

## 【NEW】はじめてログインする（招待されたら）

オーナーまたは管理者（店長）があなたを従業員として登録すると、招待メールが届きます。

### 招待メールからパスワードを設定する

1. 届いたメールの「パスワードを設定する」ボタンをタップ

> **📋 スクリーンショット必要:** 招待メール本文のイメージ（モック C-4画面からキャプチャ）

2. パスワード（8文字以上）を入力
3. 「登録を完了してログイン」をタップ

> **📋 スクリーンショット必要:** PW設定画面（モック C-5画面からキャプチャ）

4. ログイン後、スマートフォンから打刻できるようになります

### 招待メールが届かない場合

- 迷惑メールフォルダを確認してください
- それでも届かない場合は、オーナーまたは管理者（店長）に招待メールの再送信を依頼してください

### 「招待リンクが無効です」と表示された場合

> **📋 スクリーンショット必要:** 無効な招待リンクのエラー画面（モック invite-invalid画面からキャプチャ）

招待リンクの有効期限は**72時間**です。以下の原因が考えられます。

- 招待の有効期限（72時間）が切れた
- 招待が取り消された
- 既にアカウントが作成済み

**対処法:** オーナーまたは管理者（店長）に連絡し、新しい招待メールの送信を依頼してください。

## 打刻を忘れた

打刻漏れの修正は管理者のみ編集が可能です。管理者へ打刻漏れの旨を伝え、勤務時間・休憩時間を申告してください。

打刻漏れの確認は勤務記録よりご確認いただけます。

打刻漏れを通知する設定も可能です。

---

## 勤務記録確認方法

> **📋 スクリーンショット:** 勤務記録画面（既存スクショ流用）

1. 右上のアイコンクリックしてメニュー表示
2. メニュー2番目の「勤務記録」をタップ
3. 当月の勤務記録が表示（店舗名、勤務時間、ステータス）
4. 「打刻漏れのみ」タブを選択すると、退勤打刻記録がないもののみ表示（管理者に記録登録依頼）

---

## 打刻漏れ通知設定

> **📋 スクリーンショット:** 通知設定画面（既存スクショ流用）

1. 右上のアイコンクリックしてメニュー表示
2. メニュー最下部の名前をタップ
3. 「通知設定」を選択
4. 「打刻忘れ通知」のトグルをタップしてオンにすることで、退勤打刻がない状態で翌日になった場合にプッシュ通知送信

# オーナー / 本部管理者向け

## 【NEW】オーナー登録（新規会社登録）

ラクミー タイムレコードを初めてご利用される方は、オーナーアカウントを作成してください。

### アカウント作成

1. [ログインページ](https://timerecord.rakmy.jp/login)にアクセスする
2. 画面下部の「新規登録はこちら」をクリック

> **📋 スクリーンショット必要:** ログイン画面（「新規登録はこちら」リンクが見える状態。モック Login画面からキャプチャ）

3. 名前・メールアドレス・パスワード（8文字以上）を入力し「アカウントを作成」をクリック

> **📋 スクリーンショット必要:** 新規登録画面（モック A-1画面からキャプチャ）

### メール認証

4. 入力したメールアドレスに6桁の確認コードが届きます
5. 確認コードを入力し「確認する」をクリック

> **📋 スクリーンショット必要:** メール認証画面（モック A-2画面からキャプチャ）

- 確認コードの有効期限は30分です
- 届かない場合は「確認コードを再送信」をクリックしてください

### 会社情報の登録

6. メール認証が完了すると、会社情報の登録画面が表示されます
7. 会社名を入力し「登録を完了する」をクリック

> **📋 スクリーンショット必要:** 会社登録画面（モック A-3画面からキャプチャ）

### 登録完了

8. 登録完了画面が表示されます
9. 「店舗を登録する」をクリックして初期設定を開始してください

> **📋 スクリーンショット必要:** 登録完了画面（モック A-4画面からキャプチャ）

### 登録後の初期設定の流れ

| 順序 | 設定内容 | 操作場所 |
|------|---------|---------|
| 1 | 店舗登録 | 設定 → 店舗情報 → 「+ 店舗登録」 |
| 2 | 従業員登録 | 設定 → 従業員 → 「+ 従業員登録」（招待メール自動送信） |
| 3 | 管理者登録（任意） | 設定 → 管理者 → 「+ 管理者を登録」 |

## 【NEW】管理者（店長）の登録

管理者（店長）はオーナーが直接登録します。招待メールは送信されません。

> **この操作はオーナーのみ実行可能です。**

### 登録手順

1. ナビゲーションメニューから「設定」→「管理者」を選択
2. 「+ 管理者を登録」をクリック

> **📋 スクリーンショット必要:** 管理者一覧画面（モック D-1画面からキャプチャ）

3. 以下の情報を入力

| 項目 | 入力内容 |
|------|---------|
| 名前 | 管理者の氏名 |
| メールアドレス | ログインに使用するメールアドレス |
| 初期パスワード | 8文字以上 |
| 初期パスワード（確認） | 同じパスワードを再入力 |
| 担当店舗 | 管理を担当する店舗にチェック（複数選択可） |

> **📋 スクリーンショット必要:** 管理者登録モーダル（モック D-2画面からキャプチャ）

4. 「管理者を登録」をクリック

### 管理者（店長）への共有事項

登録後、管理者に以下を**口頭またはチャット**で伝えてください。

- ログインURL: https://timerecord.rakmy.jp/login
- メールアドレス
- 初期パスワード

> **注意:** 初期パスワードはメールでは送信されません。セキュリティのため口頭やチャット等で安全に共有してください。

### 管理者の編集

1. 管理者一覧で対象の「編集」をクリック
2. 名前・メールアドレス・担当店舗を変更して「保存」

> **📋 スクリーンショット必要:** 管理者編集モーダル（モック D-1E画面からキャプチャ）

### 管理者の削除

1. 管理者一覧で対象の「削除」をクリック
2. 確認ダイアログで「OK」を選択

> **注意:** 削除するとアカウントが完全に削除されます。この操作は取り消せません。

## 【NEW】データインポート（従業員一括登録）

CSVファイルを使って従業員データを一括登録できます。

> **オーナー:** 全店舗の従業員をインポート可能
> **管理者（店長）:** 担当店舗の従業員のみインポート可能

### テンプレートのダウンロード

1. ナビゲーションメニューから「設定」→「インポート」を選択
2. 「テンプレートをダウンロード」をクリック

> **📋 スクリーンショット必要:** インポート画面（テンプレートDLボタンが見える状態）

### CSVフォーマット

| 項目 | 必須 | 説明 |
|------|------|------|
| 名前 | ✅ | 従業員の氏名 |
| メールアドレス | ✅ | 招待メール送信先 |
| 電話番号 | | 任意 |
| 雇用区分 | ✅ | 登録済みの雇用区分名 |
| 交通費単位 | ✅ | 「日額」または「月額」 |
| 交通費 | ✅ | 金額（数値） |
| 所属店舗 | ✅ | 登録済みの店舗名 |
| 従業員コード | | 未入力時は自動生成 |

### インポート手順

1. ダウンロードしたテンプレートに従業員データを入力・保存
2. 「CSVファイルを選択」をクリックし、作成したファイルを選択
3. プレビュー画面でデータの内容を確認
4. 「インポートを実行」をクリック

> **📋 スクリーンショット必要:** CSVファイル選択 → プレビュー → 実行確認（2〜3枚）

### インポート結果

- **成功:** 従業員が登録され、招待メールが自動送信されます
- **エラー:** エラーのある行はスキップされ、エラー内容が表示されます

> **📋 スクリーンショット必要:** インポート結果画面（成功/エラー混在の例）

### 注意事項

- 1回のインポートで最大100件まで登録可能
- 既に登録済みのメールアドレスはエラーになります
- インポートした従業員には自動で招待メールが送信されます

# 管理者（店長）向け

## 【NEW】管理者（店長）の初回ログイン

オーナーから共有されたログイン情報でログインしてください。

1. [ログインページ](https://timerecord.rakmy.jp/login)にアクセスする
2. オーナーから共有されたメールアドレスと初期パスワードでログイン
3. セキュリティのため、ログイン後にパスワードを変更してください
    - サイドバー「アカウント設定」から変更可能

> **📋 スクリーンショット必要:** ログイン画面 + アカウント設定のパスワード変更画面

### パスワードを忘れた場合

1. ログイン画面の「パスワードを忘れた」をクリック
2. 登録済みのメールアドレスを入力
3. メールアドレス宛に確認コードが届く
4. 確認コードを入力し、新しいパスワードを設定

> **📋 スクリーンショット必要:** PW再設定フロー（モック E-1 → E-2 → E-3画面からキャプチャ）

---

## 【NEW】従業員の登録・招待

従業員を登録すると、招待メールが自動送信されます。

> **管理者（店長）は担当店舗の従業員のみ登録可能です。**

### 登録手順

1. ナビゲーションメニューから「従業員」を選択
2. 「+ 従業員登録」をクリック

> **📋 スクリーンショット必要:** 従業員一覧 管理者視点（モック C-2画面からキャプチャ）

3. 以下の情報を入力

| 項目 | 入力内容 |
|------|---------|
| 所属店舗 | 担当店舗から選択（1店舗の場合は自動設定） |
| 名前 | 従業員の氏名 |
| 電話番号 | 電話番号 |
| メールアドレス | 招待メール送信先 |
| 従業員コード | 任意（未入力時は自動生成） |
| 雇用区分 | 雇用区分を選択 |
| 交通費単位 | 「日額」または「月額」 |
| 交通費 | 金額 |

> **📋 スクリーンショット必要:** 従業員登録モーダル 管理者視点（モック C-3M画面からキャプチャ）

4. 「登録して招待メールを送信」をクリック

### 従業員のステータス

| ステータス | バッジ色 | 意味 |
|----------|---------|------|
| 招待中 | オレンジ | 招待メール送信済み、パスワード未設定 |
| 有効 | 緑 | パスワード設定完了、ログイン可能 |
| 期限切れ | 赤 | 72時間以内にパスワード設定が行われなかった |

### ステータス別の操作

| ステータス | PW操作 | 編集 | 削除 |
|----------|--------|------|------|
| 招待中 | 「再送」で招待メール再送信 | メール変更可（変更時は招待再送） | 招待を取り消し |
| 有効 | 「PW再発行」で一時PW発行（メール通知） | メール変更不可 | アカウント削除（取消不可） |
| 期限切れ | 「再送」で招待メール再送信 | メール変更可（変更時は招待再送） | 招待を取り消し |

---

## 【NEW】従業員のPW再発行

従業員がパスワードを忘れた場合、管理者（店長）がパスワードを再発行できます。

> **管理者は担当店舗の従業員のみ操作可能です。**

1. 従業員一覧で対象の従業員のPW操作列「PW再発行」をクリック
2. 確認ダイアログで「OK」を選択
3. 従業員のメールアドレスに一時パスワードが通知されます

> 従業員は一時パスワードでログイン後、パスワードを変更してください。

---

## 【NEW】招待メール再送信

招待中または期限切れの従業員に、招待メールを再送信できます。

> **管理者は担当店舗の従業員のみ操作可能です。**

1. 従業員一覧で対象の従業員のPW操作列「再送」をクリック
2. 新しい招待メールが送信されます

- 再送信は1日5回まで / 再送間隔は60秒以上

---

## 【NEW】従業員の店舗異動

担当店舗間で従業員の所属店舗を変更できます。

> **管理者は担当店舗間のみ異動可能です。担当外の店舗への異動はオーナーに依頼してください。**

1. 従業員一覧で対象の従業員の「編集」をクリック
2. 「所属店舗」のセレクトボックスで新しい店舗を選択

> **📋 スクリーンショット必要:** 編集モーダルで所属店舗変更時（⚠️交通費アラート + 赤枠ハイライト表示状態。モック C-2E画面からキャプチャ）

3. ⚠️「交通費（日額）を新しい店舗に合わせて更新してください」のアラートが表示されます
4. 交通費単位・交通費を新しい店舗に合わせて更新（赤枠でハイライトされます）
5. 「保存」をクリック

---

## 【NEW】外部連携設定（経営管理連携）

担当店舗の外部連携設定を行えます。詳細は「オーナー / 本部管理者向け」の[外部連携設定（経営管理連携）](#new外部連携設定経営管理連携)を参照してください。

> **管理者は担当店舗の外部連携設定のみ操作可能です。**

## 打刻を修正したい

### 打刻修正

ナビゲーションメニューから「打刻記録」を選択し、該当打刻の✏を選択すると打刻修正画面が表示されます。

> **📋 スクリーンショット:** 打刻記録画面（既存スクショ流用）

### 打刻漏れの修正

ナビゲーションメニューから「打刻記録」を選択し、「打刻漏れのみ」タブを選択すると退勤打刻記録がないもののみ表示されます。

> **📋 スクリーンショット:** 打刻漏れ画面 + 打刻修正モーダル（既存スクショ流用）

修正する従業員の✏を選択すると打刻修正画面が表示されます。
