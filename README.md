# rakmy-owner-registration-mock

ラクミータイムレコード オーナー登録・店舗管理・権限管理 要件書/モック/検証/CS申し送り一式。

> **制作:** [@suguru789987](https://github.com/suguru789987) (PdM)
> **担当範囲:** 権限階層3層(オーナー/管理者/従業員)設計 / 権限マトリクス / UIモック(v1〜v5) / UIロジック詳細(交通費ハイライト等) / CS申し送り表 / ヘルプページIA再設計 / FAQ整備
> **ツール:** Claude Code / HTML / CSS / Markdown / TSV

---

## ハイライト

- オーナー → 管理者 → 従業員の**3階層権限マトリクス**を設計
- プロダクト内用語を「**店長**」→「**管理者**」に統一する命名ルール断行
- 招待メール**72時間有効期限**、PW8文字以上の仕様
- **UIモック5バージョン**(v1.0〜v5) を並存管理
- **CS申し送り表** 7ファイルを役割別/ケース別に整備
- **ヘルプページ全面改訂**: 役割別(オーナー/管理者/従業員) × ケース別 のIA再設計
- **FAQ整備**: CS問い合わせ傾向から項目化
- **費用実績入力モーダル**(地代家賃実績 等) の独立モックを追加
