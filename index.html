ラクミー タイムレコード — 仕様モック v5（エンジニア共有用）

開発対象の画面のみ抜粋。全画面は v4（開発用）を参照。

v5 に含まれる画面:
・A 新規登録: 入力 → メール認証 → PW設定 → 完了
・C 従業員管理: オーナー視点一覧 / 管理者視点一覧 / 招待メール（自動送信） / 従業員PW設定
・D 管理者管理: 管理者一覧（PW操作列あり・ステータス列なし）
・E ログイン / PW再設定: ログイン → メール入力 → 認証 → 新PW設定

初期セットアップ（必須）

タイムレコードを利用開始するには、以下の順番で全項目の登録が必要です。
1
業態・ブランド登録 進行中
店舗の業態（飲食店・小売店など）と運営ブランド名を登録します。
前提登録:なし（独立して登録可能）
登録済み: 0 件
2
店舗登録 🔒 ①完了後
店舗名・住所・営業時間等の店舗情報を登録します。複数店舗の登録も可能です。
前提登録:① 業態・ブランド
登録済み: 0 件
3
雇用区分登録 🔒 ②完了後
正社員・アルバイト・業務委託など、従業員の雇用区分を登録します。
前提登録:なし（独立して登録可能）
登録済み: 0 件
4
従業員登録 🔒 ③完了後
店舗・雇用区分を割り当てて従業員を登録します。招待メールは登録後、従業員一覧の「招待送信」ボタンから送信できます。
前提登録:② 店舗③ 雇用区分
登録済み: 0 件
5
管理者登録 任意・④完了後
担当店舗の従業員管理・勤怠管理を委任する管理者を登録します。必須ではありません。新規入力または既存従業員からの選択で登録できます。
