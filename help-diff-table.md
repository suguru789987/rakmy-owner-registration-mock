# UIロジック・ヘルプページ差分表（インポート機能 + セルフ登録機能）

## 1. UIロジック差分

### ログイン・認証まわり

| 画面/機能 | 実装前（現行） | 実装後 | CS申し送り |
|----------|-------------|--------|----------|
| ログイン画面 | メール + パスワードのみ | 「新規登録はこちら」リンク追加、「パスワードを忘れた」リンク追加 | 「新規登録はこちら」からオーナー登録が可能になった旨を案内可能に。従来のCLI登録依頼は不要 |
| オーナー登録 | 開発者がCLIで手動登録 | セルフサービス（アカウント作成→メール認証→会社登録→完了） | CLIでの登録依頼対応が不要に。「ログイン画面→新規登録はこちら」を案内 |
| メール認証 | なし | 6桁確認コード（30分有効、再送可能） | 「コードが届かない」問い合わせ → 迷惑メールフォルダ確認を案内。再送ボタンあり（60秒間隔） |
| PW再設定 | 管理者に連絡して再設定 | ログイン画面「パスワードを忘れた」から自分でリセット可能 | 「パスワードを忘れた」→ 自分でリセットできる旨を案内。CS側でのPWリセット対応が減少 |

### 管理者（店舗管理者）まわり

| 画面/機能 | 実装前（現行） | 実装後 | CS申し送り |
|----------|-------------|--------|----------|
| 管理者の登録 | なし（開発者が手動設定） | オーナーが管理者一覧から直接登録（初期PW設定→口頭共有） | 管理者登録はオーナー自身で可能。初期PWはメール送信されないため「PWが届かない」は仕様通り。口頭/チャット共有を案内 |
| 管理者一覧 | なし | サイドバー「設定」→「管理者」に追加。名前・メール・権限・担当店舗・操作列 | PW操作列なし（管理者は自分でPW再設定）。「管理者のPWをリセットしたい」→ ログイン画面の「パスワードを忘れた」を案内 |
| 管理者の編集 | なし | 名前・メール・担当店舗を編集可能 | 担当店舗の変更はオーナーのみ可能 |
| 管理者の削除 | なし | オーナーが削除（確認ダイアログあり、取消不可） | 削除するとアカウントが完全削除される。復元不可の旨を案内 |
| 管理者のPW忘れ | CS/開発者対応 | 管理者自身がログイン画面「パスワードを忘れた」からリセット | CS対応不要。自己解決を案内 |

### 従業員管理まわり

| 画面/機能 | 実装前（現行） | 実装後 | CS申し送り |
|----------|-------------|--------|----------|
| 従業員一覧（オーナー） | 基本的な一覧表示 | 店舗フィルタータブ + 所属店舗列 + 3ステータス（有効/招待中/期限切れ）表示 + PW操作列 | ステータスの意味: 招待中=メール送信済み未設定、有効=PW設定完了、期限切れ=72時間超過 |
| 従業員一覧（管理者） | なし（管理者機能自体がなかった） | 担当店舗の従業員のみ表示。オーナーと同じステータス別ボタン | 管理者は担当店舗の従業員のみ表示される。他店舗は見えない仕様 |
| 従業員登録（オーナー） | 基本登録のみ | 登録と同時に招待メールが自動送信。所属店舗セレクト（全店舗） | 「登録したのにログインできない」→ 従業員がメールからPW設定を完了する必要がある旨を案内 |
| 従業員登録（管理者） | なし | 担当店舗のみセレクト。1店舗の場合はdisabled | 管理者は担当店舗の従業員のみ登録可能 |
| 従業員編集（オーナー） | 基本編集 | 有効: メール変更不可。招待中/期限切れ: メール変更可（変更時は招待再送）。所属店舗変更時は交通費アラート+赤枠ハイライト | 「メールアドレスを変更したい（有効）」→ 削除して新メールで再登録を案内 |
| 従業員編集（管理者） | なし | オーナーと同様。所属店舗は担当店舗のみ（店舗間異動対応）。交通費アラート+赤枠ハイライト | 管理者は担当店舗間でのみ異動可能。担当外への異動はオーナーに依頼 |
| PW再発行 | CS/開発者対応 | オーナー/管理者が従業員一覧のPW操作列から実行。一時PWがメール通知 | 「従業員がPWを忘れた」→ オーナーまたは管理者にPW再発行を依頼するよう案内 |
| 招待メール再送信 | なし | 招待中/期限切れの従業員に対して「再送」ボタンで再送信 | 「招待メールが届かない」→ オーナー/管理者に再送を依頼。迷惑メールフォルダも確認 |
| 招待リンク期限切れ | なし | 72時間で期限切れ。エラー画面「招待リンクが無効です」を表示 | 「招待リンクが無効」→ オーナー/管理者に再送信を依頼するよう案内 |

### インポート機能

| 画面/機能 | 実装前（現行） | 実装後 | CS申し送り |
|----------|-------------|--------|----------|
| 従業員一括登録 | 1件ずつ手動登録 | CSVインポートで最大100件一括登録。招待メールも自動送信 | インポートエラー時はエラー行がスキップされ結果画面に表示。テンプレートDL機能あり |
| 店舗一括登録 | 1件ずつ手動登録 | CSVインポートで一括登録（オーナーのみ） | 店舗インポートはオーナー限定。管理者には権限なし |
| テンプレート | なし | インポート画面からCSVテンプレートをダウンロード可能 | 「どのフォーマットで作ればいいか」→ テンプレートDLを案内 |
| インポート権限 | なし | オーナー: 全店舗 / 管理者: 担当店舗の従業員のみ | 管理者は担当店舗の従業員インポートのみ可能 |

### 店舗管理まわり

| 画面/機能 | 実装前（現行） | 実装後 | CS申し送り |
|----------|-------------|--------|----------|
| 店舗一覧 | 基本的な一覧表示 | 操作列に「QR」「編集」「削除」ボタン（アイコン）。管理者列なし | 店舗の操作はオーナーのみ可能。管理者は店舗の追加・編集・削除不可 |
| 店舗追加 | 基本登録 | 店舗名・業態・郵便番号・住所のみ（管理者設定は管理者一覧で行う） | 管理者の店舗割り当ては「設定→管理者→編集」から行う |

## 2. ヘルプページ差分

### 従業員アプリ（スマホ）

| ヘルプ項目 | 実装前 | 実装後 | CS申し送り |
|----------|--------|--------|----------|
| パスワードを忘れた | ✅ 掲載済み | ✅ 変更なし（既存のまま） | 変更なし |
| 勤務記録確認方法 | ✅ 掲載済み | ✅ 変更なし | 変更なし |
| 打刻漏れ通知設定 | ✅ 掲載済み | ✅ 変更なし | 変更なし |
| 招待メールからのPW設定 | なし | **新規追加** — 招待メール受信→リンクからPW設定→ログインの手順 | 「登録されたがログインできない」→ 招待メールからPW設定が必要な旨を案内 |
| 招待リンクが無効 | なし | **新規追加** — エラー画面の説明と対処法（管理者に再送依頼） | 72時間で期限切れ。「無効です」が出たらオーナー/管理者に再送依頼を案内 |

### 管理者ダッシュボード（PC）

| ヘルプ項目 | 実装前 | 実装後 | CS申し送り |
|----------|--------|--------|----------|
| パスワードを忘れた | ✅ 掲載済み | ✅ 変更なし（ログイン画面から自己リセット） | 変更なし |
| 打刻を修正したい | ✅ 掲載済み | ✅ 変更なし | 変更なし |
| オーナー登録（新規） | なし | **新規追加** — セルフサインアップ手順（アカウント作成→メール認証→会社登録→完了） | CLI登録依頼が不要に。「新規導入したい」→ ログイン画面の「新規登録はこちら」を案内 |
| 管理者登録 | なし | **新規追加** — オーナーによる直接登録手順（PW設定→口頭共有→初回ログイン） | 管理者のPWはメール送信されない。口頭共有が必要な旨を案内 |
| 管理者の編集・削除 | なし | **新規追加** — 管理者一覧からの編集・削除手順 | 削除は取消不可。権限変更（担当店舗の追加/削除）は編集から |
| 管理者ができること | なし | **新規追加** — 管理者の権限一覧表 | 管理者は担当店舗のみ操作可能。「他店舗が見えない」は仕様通り |
| 従業員登録（招待） | なし | **新規追加** — 登録→招待メール自動送信→PW設定の流れ | 「登録したのにログインできない」→ 従業員がメールからPW設定を完了する必要がある |
| 従業員ステータス | なし | **新規追加** — 招待中/有効/期限切れの意味と操作 | ステータスの違いと各ステータスでできる操作を案内 |
| PW再発行 | なし | **新規追加** — 従業員のPWを管理者がリセットする手順 | 一時PWがメール通知される。次回ログイン時に変更推奨 |
| 招待メール再送信 | なし | **新規追加** — 招待中/期限切れの従業員への再送手順 | 再送は5回/日まで。60秒間隔 |
| 従業員の店舗異動 | なし | **新規追加** — 編集モーダルでの所属店舗変更手順。交通費更新アラートの説明 | 異動時は交通費の更新を忘れないよう注意喚起 |
| データインポート | なし | **新規追加** — CSVテンプレートDL→データ入力→インポート実行→結果確認 | テンプレートDL可能。エラー行はスキップされ結果表示。最大100件/回 |

## 3. CS問い合わせ想定FAQ（新機能関連）

| 想定問い合わせ | 回答方針 | 該当ヘルプ |
|-------------|---------|----------|
| 「新規で利用開始したい」 | ログイン画面「新規登録はこちら」からセルフ登録を案内。CLI依頼は不要 | オーナー登録 |
| 「管理者のPWがわからない」 | ログイン画面「パスワードを忘れた」から自己リセットを案内 | PW再設定 |
| 「従業員を登録したがログインできない」 | 従業員にメールからPW設定を完了するよう案内 | 従業員登録（招待） |
| 「招待メールが届かない」 | 迷惑メールフォルダ確認→オーナー/管理者に再送依頼 | 招待メール再送信 |
| 「招待リンクが無効と表示される」 | 72時間で期限切れ。オーナー/管理者に再送依頼 | 招待リンクが無効 |
| 「管理者に他店舗が見えない」 | 仕様通り。担当店舗のみ表示。他店舗はオーナーに依頼 | 管理者ができること |
| 「従業員のメールアドレスを変更したい（有効）」 | 有効ステータスでは変更不可。削除→新メールで再登録を案内 | 従業員編集 |
| 「インポートでエラーが出た」 | エラー行はスキップされ結果画面に表示。テンプレートに沿って修正し再インポート | データインポート |
| 「初期パスワードがメールで届かない（管理者）」 | 仕様通り。管理者のPWはメール送信されない。オーナーから口頭/チャットで共有 | 管理者登録 |
| 「店舗を異動させたら給与はどうなるか」 | 打刻ベースで計算。異動前の打刻は旧店舗、異動後は新店舗に計上 | 従業員の店舗異動 |
